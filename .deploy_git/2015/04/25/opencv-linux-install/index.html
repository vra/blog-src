这篇博客介绍在Linux中的gcc和g++编译环境下如何使用cmake来编译OpenCV源代码。我基本是按照OpenCV官方的说明文档，一步步地进行的，所以表述不清楚的地方还请参照原文。

1. 编译环境

  • 操作系统：Ubuntu 14.10
  • gcc 版本: 4.9.1
  • cmake 版本： 2.8.12.2
  • opencv版本： 2.4.10
